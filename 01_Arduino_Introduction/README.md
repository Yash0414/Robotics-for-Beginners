# Lesson 01 - Arduino Introduction

## Objective

Learn what Arduino is, where it is used, and how it helps in robotics and IoT projects.

---

## What is Arduino?

Arduino is an open-source microcontroller platform that allows us to control electronic devices using code.

It is commonly used in:

* Robotics
* Home Automation
* IoT Projects
* Smart Agriculture
* STEM Education

---

## Arduino Uno Overview

### Main Components

* Microcontroller
* USB Port
* Digital Pins
* Analog Pins
* Power Pins
* Reset Button

---

## Why Learn Arduino?

* Beginner Friendly
* Large Community Support
* Low Cost
* Industry Standard for Prototyping

---

## Applications

* Robot Cars
* Smart Home Systems
* Weather Stations
* Security Systems
* Automation Projects
